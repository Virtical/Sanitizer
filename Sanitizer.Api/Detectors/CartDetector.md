Описание: Система должна обнаруживать номера банковских карт (PAN) всех основных платёжных систем с использованием алгоритма Luhn для валидации.
Критерии приёмки:
•	Распознаются номера карт Visa
•	Распознаются номера карт MasterCard
•	Распознаются номера МИР
•	Не распознаются последовательные числа типа 1234567890123456
•	Распознаются номера в различном контексте текста
