Описание: Система должна обнаруживать распространённые форматы API-ключей, токенов доступа и секретов на основе паттернов.
Критерии приёмки:
•	Распознаются ключи в формате: sk-XXXXXXXXXXXXXXXXXXXX (OpenAI)
•	Распознаются GitHub токены: ghp_XXXXXXXXXXXXXXXXXXXX, gho_XXXXXXXXXXXXXXXXXXXX
•	Распознаются ключи с префиксами (api_key=, token=, secret=)
•	Настраиваемый список паттернов для кастомных API
