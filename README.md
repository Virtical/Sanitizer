# Sanitizer

**Автоматическая анонимизация чувствительных данных** — решение для защиты PII и конфиденциальной информации в промптах, логах, API и кодовой базе.

## Актуальность

Компании всё чаще сталкиваются с рисками утечек чувствительных данных через:
- LLM-промпты и чат-боты
- Логи и отладочную информацию
- API-запросы
- Кодовую базу и бизнес-логику

Сотрудники иногда вставляют реальные данные в публичные модели, что приводит к «тихим» утечкам.

## Возможности

- **Детекция PII** различных типов
- **Гибкие стратегии маскирования** (в т.ч. обратимая токенизация)
- **Web-чаты** с настраиваемой санитизацией и историей
- **OpenAI-совместимый Proxy API** (с ограничением RPS)
- **REST API** и **SDK для .NET**
- **Web UI** для управления
- Настраиваемые профили и плагинная архитектура

## Сравнение с конкурентами

Sanitizer выгодно отличается наличием обратимой токенизации, LLM Proxy, .NET SDK, Web UI и полной плагинной архитектуры.

## Технологический стек

- **Blazor** + **.NET 8** (C# 12)
- HTML / CSS / JavaScript

**Преимущества**: высокая производительность, полная прозрачность асинхронности, единая модель данных, быстрая компиляция.
